Déployez en une commande
Un docker-compose.yml prêt à l'emploi. Fonctionne sur n'importe quel serveur Linux, Raspberry Pi ou homelab Proxmox.
Docker
# Clonez et lancez
git clone https://github.com/aurel-f/pooly && cd pooly
cp .env.example .env && docker compose up -d

Lancez Pooly
en 3 minutes

Un docker-compose prêt à l'emploi. Aucune dépendance à installer en dehors de Docker.

1
Clonez le dépôt
Récupérez le code depuis GitHub en une commande.
2
Configurez l'environnement
Copiez .env.example et définissez vos mots de passe.
3
Lancez avec Docker Compose
L'app, l'API et la base de données démarrent en une seule commande.
Voir sur GitHub Lire la documentation →
terminal
# 1. Clonez le dépôt
git clone https://github.com/aurel-f/pooly
cd pooly

# 2. Configurez l'environnement
cp .env.example .env
nano .env

# 3. Lancez Pooly
docker compose up -d

# ✓ Accessible sur http://localhost:8090
